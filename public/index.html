Football Passing Networks

A passing network is just the application of network theory and its analysis tools to understand the performance of a football team in a match.

Each player is represented by a vertex. Each vertex is connected with many others by edges, which represent the passes exchanged between players.

Down below you can check out the passing networks of several important championships. So far we have available the 2018 World Cup, Italian 2018, La Liga 2018, Italian 2017, La Liga 2017 and UEFA Euro 2016.
